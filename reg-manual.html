Регистрация доменов

По одному на основной whois

  1. Заказ услуги → Регистрация доменов
  2. Вводим доменное имя, отмечаем галочкой нужную зону. Если хотим зарегистрировать сразу несколько доменов - кликаем по ссылке «Перейти в режим массовой регистрации»
  3. Изменяем если нужно мыло и номер телефона для whois
  4. Прописываем NS-сервера
  5. Если нужно - можно скрыть персональные данные в whois
  6. Заказать

Массовая регистрация на разные whois

  1. Сходить в генератор whois и создать файл в формате почтового API. Забить нужные домены, указать NSы если нужно. Полученный файл Сохранить как...
    Осторожно! Изменяя этот файл, помните, что он должен оставаться в кодировке kio-8, иначе робот регистратора нифига не разберет
  2. В панели: Для реселлеров → Домены → Массовая регистрация доменов
  3. Выбрать сохраненный файл из генератора

Внимательно полазив по админке можно найти ответы на все свои вопросы. Если же всё-таки вопросы остаются - вэлком в аську.
